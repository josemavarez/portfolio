ENCUENTRAME
2021

Diseño de Interfaz y Experiencia

El diseño de interfaz de usuario aplicado a Codelin tiene como objetivo hacer accesibles los elementos que permitan una navegación orientada a las conversiones. Se realizan pruebas A/B continuas para definir futuros cambios en base a la experiencia.

Prioridad de navegación efectiva.

El flujo de usuarios incluye secciones que en cada punto de contacto guían efectivamente al visitante a encontrar el producto que busca, todas las subcategorías fueron agregadas a la lista de productos con una lógica de radio button que permite activar o desactivar los filtros de una manera simple.

También se añaden apartados “Multifunción” donde se ofrecen productos mediante venta cruzada y se garantiza la obtención de leads a través de diferentes áreas de suscripción.

2018

Interfaz de usuario de diseño de aplicación de comida de entrega

A partir de un estudio de mercado se estableció el orden de los elementos en la interfaz, favoreciendo su accesibilidad de forma sencilla e intuitiva.

Diseño de experiencia cómodo y sencillo.

2020

Email Marketing UI

Enfocado en enviar un mensaje con el fin de adquirir nuevos clientes, desarrollar relaciones con los actuales, fidelizar, interactuar con contactos, aumentar las ventas, generar confianza hacia un servicio o producto, confirmar una orden de compra, entre otros objetivos.

Campañas efectivas

Cuando se usa correctamente, la conversión de una campaña de emailing es mucho mayor que en las redes sociales, por lo que es una de mis herramientas favoritas.

Lo considero uno de los canales más rentables del marketing online, pero es necesario tener una estrategia de contenidos clara, concisa y que aporte valor.

2019

Diseño UI Ecommerce

Uno de los aspectos más importantes a considerar al diseñar la interfaz de usuario es la navegación. Esta debe ser fluida e intuitiva, para que los visitantes puedan navegar sin dificultad por todas las áreas del sitio web.

Interfaz intuitiva

Si la interfaz es complicada de usar, el prospecto no se sentirá seguro mientras navega y no realizará la compra. Por ello, es importante ofrecer a los visitantes una experiencia agradable desde la página principal, incluyendo -por ejemplo- una barra de opciones que permita avanzar o retroceder sin dificultad.

2019

Estructura ordenada

Los elementos del sitio web deben estar distribuidos de manera ordenada y armoniosa. Hoy en día, el consumidor es más exigente, por lo que un mal diseño puede generar desconfianza hacia la marca. Para contrarrestar este problema, accede a botones para mejorar la navegación dentro de su sitio web. Además, les permitirá enlazar su contenido, agilizando así el proceso de compra.

Diseño responsivo

Hoy en día, las personas acceden a sitios web desde diferentes computadoras. Por ello, es fundamental que las plataformas se adapten automáticamente al tamaño de los distintos dispositivos para garantizar la preferencia del usuario. Por lo tanto, se logró que la interfaz tenga un diseño simple y directo para mantener una estructura amigable.

2020

Immersive user experience

Optimización de la experiencia integral y el conjunto de interacciones que los usuarios tienen con una aplicación en dispositivos portátiles, incluidos teléfonos inteligentes, tabletas y dispositivos portátiles.

Proceso de diseño y desarrollo

Para aclarar, el diseño de UX y UI van de la mano en cualquier proceso de diseño. UX se trata de experiencia: el viaje holístico de los puntos de contacto en el diseño de productos. La UI se ocupa de la interfaz: el lenguaje de diseño y el sistema de elementos con los que interactúa el usuario.
