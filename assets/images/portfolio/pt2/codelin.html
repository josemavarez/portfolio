NAVEGACIÓN
El diseño de interfaz de usuario aplicado a Codelin tiene como objetivo hacer accesibles los elementos que permitan una navegación orientada a las conversiones. Se realizan pruebas A/B continuas para definir futuros cambios en base a la experiencia.
